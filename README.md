# Starship Test

A test repository for validating [Starship](https://starship.rs/) prompt configuration.

## Stack

- Python 3.12+
- Node.js (tooling)
- Docker (deployment)

## Features

- Git status color-coding (staged/modified/untracked)
- Remote origin display with platform icon
- Branch count tracking
- Push/pull indicators
- Language & package version detection
